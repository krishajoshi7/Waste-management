# Sustainable ECG Prototype

Responsive Next.js prototype for textile waste pickup operations.

## What is included

- Generator flow: home, waste logging, photo placeholders, pickup tracking.
- Collector flow: assigned pickups, pickup detail, completion form, warehouse delivery form.
- Admin flow: operational metrics and request assignment table.
- Mock data in `app/prototype-data.ts`.
- Product docs in `docs/`.

## Run Locally

Install dependencies, then start the development server:

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```
